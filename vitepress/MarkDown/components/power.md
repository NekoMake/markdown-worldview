# Power - 势力综测

::: warning 开发中
此组件正在开发中，预计在 Phase 4 完成。
:::

## 组件介绍

Power 组件用于展示势力或组织的综合状态，包括数值统计、趋势分析等。

## 计划功能

基于 ECharts 柱状图 + 纯 CSS 混合实现，支持：

- 📊 多维度数值展示
- 📈 趋势标识（上升/稳定/下降）
- 🎯 状态颜色映射（战争/和平/中立）
- 💡 领袖信息展示

## YAML 语法（规划中）

````markdown
```power
faction: 凛冬帝国
leader: 亚历山大三世
status: 战争状态
data:
  经济实力: [80, "产出：魔导矿石"]
  军事实力: [95, "常备军：12万"]
  人口规模: [65, "总人口：450万"]
trend:
  经济: rising
  军事: stable
  外交: falling
```
````

## 布局结构

```
┌─────────────────────────┐
│  凛冬帝国 (战争状态)      │ ← 状态颜色高亮
│  领袖：亚历山大三世       │
├─────────────────────────┤
│  [ECharts 柱状图]        │
├─────────────────────────┤
│  趋势：                   │
│    经济 ↑  军事 →  外交 ↓ │
└─────────────────────────┘
```
