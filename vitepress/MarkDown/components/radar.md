::: warning 开发中

:::

## 组件介绍

Radar 组件用于展示多维度属性对比，适合角色能力、势力综合实力等场景。

## 计划功能

基于 ECharts 实现，支持：

- ✨ 多维度数据可视化
- 📊 自定义最大值范围
- 🎨 主题颜色自适应
- 📱 响应式图表大小

## YAML 语法（规划中）

````markdown
```radar
title: 阿尔忒尼斯
data:
  武力: 98
  敏捷: 92
  神秘: 45
  谋略: 72
  幸运: 60
```
````
